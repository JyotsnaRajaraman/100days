Project6

Program (written in Python) is an Assembler for the Hack computer that will read in a .asm file and produce a .hack file that will run on the CPU emulator.

The driver.py file is the main module that can be run from the command line by using the python command as well as providing an input file name as argument. The .hack file will be placed in the directory ion which the source .asm file resides.

For example to run a file from the terminal, you can use 

python driver.py Max.asm
