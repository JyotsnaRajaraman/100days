Program (written in Python) is the first part of the Virtual Machine Language Translator, focusing on the implementation of the stack arithmetic and memory access commands of the VM language

The driver.py file is the main module that can be run from the command line by using the python command as well as providing an input .vm file as argument. The .asm file will be placed in the directory in which the source .vm file resides.

For example to run a file from the terminal, you can use 

python driver.py PointerTest.vm
